# 🎨 Концепт-дизайн интерфейса: Constructure

## 🌟 Общая концепция дизайна

### Философия дизайна
- **Минимализм с функциональностью**: Чистый интерфейс, не отвлекающий от творческого процесса
- **Интуитивность**: Drag & drop как основной способ взаимодействия
- **Гибкость**: Множественные способы выполнения одной задачи
- **Визуальная иерархия**: Четкое разделение рабочих зон и инструментов

## 🎭 Анимации и переходы

### Переключение режимов
```css
/* Плавный переход между режимами визуализации */
.layout-transition {
  transition: all 0.6s cubic-bezier(0.4, 0, 0.2, 1);
}

/* Анимация появления узлов */
.node-enter {
  opacity: 0;
  transform: scale(0.8) translateY(20px);
}

.node-enter-active {
  opacity: 1;
  transform: scale(1) translateY(0);
  transition: all 0.3s ease-out;
}
```

### Горячие клавиши
```
Ctrl + S     - Сохранить курс
Ctrl + Z     - Отменить действие
Ctrl + Y     - Повторить действие
Ctrl + C     - Копировать выбранные узлы
Ctrl + V     - Вставить узлы
Delete       - Удалить выбранные элементы
Space        - Панорамирование (с зажатой кнопкой мыши)
Ctrl + 1-4   - Переключение режимов (Дерево, LEGO, Mind Map, Flowchart)
F            - Подогнать к размеру экрана
Esc          - Сбросить выделение
Tab          - Навигация по элементам
```

### ARIA разметка
```html
<!-- Основная рабочая область -->
<div role="application" aria-label="Конструктор курсов">
  
  <!-- Узел курса -->
  <div role="button" 
       tabindex="0"
       aria-label="Урок: Переменные в JavaScript"
       aria-describedby="node-details">
    
    <!-- Контекстное меню -->
    <div role="menu" aria-hidden="true">
      <div role="menuitem">Редактировать</div>
      <div role="menuitem">Удалить</div>
    </div>
  </div>
  
  <!-- Панель свойств -->
  <aside role="complementary" aria-label="Свойства элемента">
    <form aria-label="Настройки узла">
      <label for="node-title">Название узла</label>
      <input id="node-title" type="text" />
    </form>
  </aside>
</div>
```
